Relatório de Execução :

Para executar o projeto e os testes automatizados, os seguintes passos devem ser realizados:

1.	Clonar o repositório do projeto.

2.	Instalar as dependências com:
npm install

3.	Instalar o Playwright:
npx playwright install

4.	Executar a aplicação Angular:
ng serve

5.	Rodar os testes automatizados:
npx playwright test
